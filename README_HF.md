# ResQAI — AI Disaster Response & Relief Coordinator

An Agentic AI + RAG platform for disaster response and emergency assistance.

## Setup

Add these secrets in your Space settings:
- `GEMINI_API_KEY`: Your Google Gemini API key
- `OPENWEATHER_API_KEY`: Your OpenWeatherMap API key (optional, demo mode works without it)
